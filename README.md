# Software Project
Under the Causal Inference in Public Health Research (CIPHR) Lab at the University of South Florida, our team evaluated and compared causal mediation capabilities across various software platforms including R, SAS, Stata, and Mplus. Among all included packages, we applied causal inference models to real-world data assessing the relationship between Hispanic ethnicity and dementia outcomes, using years of education as a mediator. Technical memos were created to document findings and guide in the selection and use of mediation software for scientific research. 

This repository includes my personal contributions to the Software Project; Namely, scripts for the CMAverse and mediation packages in R.
